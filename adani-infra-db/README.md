## 🗂️ Data Structure

Each project entry contains:

| Field | Description |
|-------|-------------|
| `Sector` | Business sector (Ports, City Gas, Airports, etc.) |
| `Project/Location` | Project name or location |
| `State` | Indian state or country |
| `Year` | Year of establishment/acquisition |
| `Status` | Operational, Under Construction, Proposed |
| `GOI_Support_Type` | Classification of government support |
| `Support_Note` | Details about the support mechanism |
| `Modi_Era` | Yes/No for 2014 onwards |
| `Latitude/Longitude` | GPS coordinates |
| `Details` | Additional project information |
| `Source_URLs` | Primary source links |
| `Source_Note` | Verification guidance |

## 📋 GOI Support Classification

Projects are classified based on objective criteria:

| Category | Definition |
|----------|------------|
| Direct Govt Contract/PPP | Won through government tender or PPP bidding |
| Regulatory License (PNGRB/AAI/etc) | Obtained through regulatory authorization |
| State Government MoU | Established via MoU with state government |
| PSU Partnership/JV | Joint venture with public sector undertaking |
| Govt Land Allocation/Lease | Built on government-allocated land |
| Policy/Regulatory Benefit | Coal linkage, Environmental clearance, SEZ status |
| SECI Solar Contract | Won through SECI competitive bidding |
| Defence Production License | Licensed for defence manufacturing |
| Private Acquisition (No GOI) | Acquired through private transaction |
| Organic Growth (No GOI) | Internal business expansion |
| International (Foreign Govt) | Projects in foreign countries |
