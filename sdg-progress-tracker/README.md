---

## About

The India SDG Progress Tracker provides a comprehensive, visual overview of India's progress against the United Nations' 17 Sustainable Development Goals. Drawing on official Voluntary National Reviews, NITI Aayog SDG India Index data, and international monitoring frameworks, the tracker makes India's SDG story accessible for practitioners, researchers, policymakers, and educators.

---

## Features

| # | Feature | Description |
|---|---------|-------------|
| 1 | **All 17 SDGs** | Individual score cards and trend data for every goal |
| 2 | **Radar Charts** | Visual comparison of performance across SDG categories |
| 3 | **Trend Analysis** | Year-on-year progress tracking with trend indicators |
| 4 | **Category Breakdowns** | Grouped views: People, Planet, Prosperity, Peace, Partnership |
| 5 | **State Comparisons** | Selected state-level SDG index comparisons |
| 6 | **Interactive Charts** | Chart.js visualisations with hover detail and tooltips |
| 7 | **Responsive Design** | Works across desktop, tablet, and mobile |

## Data Sources

| Source | Coverage |
|--------|----------|
| [NITI Aayog SDG India Index](https://sdgindiaindex.niti.gov.in/) | State and national composite scores |
| [UN SDG Global Database](https://unstats.un.org/sdgs/dataportal) | International indicator data |
| [India's Voluntary National Review (VNR)](https://sustainabledevelopment.un.org/memberstates/india) | Official government reporting |
| [World Bank SDG Atlas](https://datatopics.worldbank.org/sdgatlas/) | Comparative international data |
