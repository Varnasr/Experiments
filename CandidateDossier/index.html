What This Is & Why It Matters

This tool transforms structured CSV data into professional PDF candidate dossiers with LaTeX-quality typography, using a professional blue and gold color palette optimized for government documentation. All phone numbers and emails are mandatorily masked for privacy protection.

Why this tool is necessary: Political organizations often rely on inconsistent Excel sheets, handwritten forms, or poorly formatted Word documents for candidate assessment. This leads to missing information, inconsistent evaluation criteria, and unprofessional documentation during critical selection processes.

Required CSV columns (18 fields):
preference, name, category, caste, age, gender, occupation, education, year_joined, phone, email, criminal_record, rationale, strengths, weaknesses, proposers, positions_held, elections_contested

Technical Features & PDF Output Quality

Professional PDF Features:

  • 📐 LaTeX-quality typography - Times New Roman body text with Arial headers for optimal readability
  • 📊 Structured tabular layout - Information presented in bordered tables with alternating row colors
  • 📄 One candidate per page - Easy filing and distribution for committee review
  • 🎨 Professional color scheme - Deep blue (#1e3a8a) for headers, gold (#fbbf24) for accents
  • 📏 A4 print optimization - Precise margins (20mm) and spacing for standard printing
  • 🔢 Automatic page numbering - Sequential page numbers for document organization
  • 📝 Section formatting - Clear visual hierarchy with underlined headers and justified text

Technical Implementation: Pure client-side JavaScript with zero dependencies. Uses CSS print media queries for PDF generation, ensuring consistent output across browsers. CSV parsing handles edge cases including quoted fields, commas within values, and various line endings (CRLF/LF). UTF-8 encoding support ensures proper rendering of Hindi and regional language text.
