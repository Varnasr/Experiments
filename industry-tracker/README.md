# Industry Conglomerates Database

Live at **https://industrygroups.impactmojo.in/**

Open-source geographic database of major Adani and Ambani infrastructure
projects (2001–2025) — searchable, filterable, with government-support
classification. Companion pages:

- `crony_tracker.html` — investigative tracker
- `adani_research_map.html` — research map view
- `adani_database.csv` / `.xlsx` — raw dataset
